## Docker Python Analytics (base image)

This is a base image for Python with specific analytics libraries, like:
- SciPy
- SkLearn
- NumPy
- Panda
- Statsmodels
- Flask
- PrometheusClient
- Py4J
- FindSpark
- etc...

## Build the image
```console
$ docker build -t alliander/python-analytics:0.1 .
```

## Push the image
```console
$ docker push alliander/python-analytics:0.1
```
